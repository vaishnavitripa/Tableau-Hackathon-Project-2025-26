# Fashion Retail Churn & Predictive Analytics using Tableau

## Overview
This project is a Tableau-powered analytics solution designed to help fashion and luxury e-commerce brands reduce customer churn and improve product strategy by combining sales analytics, predictive insights, and modern business workflows.

The solution demonstrates how Tableau dashboards can be integrated into collaboration and CRM environments such as Slack and Salesforce to transform insights into real-time business actions.

---

## Problem Statement
Fashion and luxury e-commerce brands face two critical challenges:
1. Identifying customers at risk of churn before revenue is lost.
2. Understanding which product categories, styles, and trends will drive future growth.

Traditional dashboards provide insights but often fail to integrate directly into day-to-day decision-making workflows.

---

## Solution
This project introduces a slide-style Tableau dashboard experience that:
- Identifies high-risk churn segments using customer behavior analytics.
- Surfaces predictive insights around fashion trends and category performance.
- Demonstrates how insights can trigger real-time alerts and actions via Slack.
- Shows conceptual integration with Salesforce CRM workflows.

---

## Key Features
- Customer churn risk segmentation (RFM-based logic)
- Predictive trend analysis using time-series insights
- Fashion category and product performance analytics
- Slide-style storytelling dashboards for executives
- Slack-based alerting for actionable insights
- Conceptual Salesforce CRM integration

---

## Architecture Overview
The solution follows a modern analytics architecture:

1. **Data Ingestion Layer**
   - Simulated e-commerce transactions, customer profiles, and fashion product data
   - Python-based data preparation and exploratory analysis

2. **Analytics Layer (Tableau Cloud)**
   - Centralised dashboard publishing and insight generation
   - Predictive analytics and storytelling dashboards

3. **Activation Layer**
   - Slack alerts triggered by key business signals
   - Conceptual Salesforce CRM embedding for decision workflows
